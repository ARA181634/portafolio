API (Application Programming Interface): Una API es un conjunto de reglas y protocolos que permiten que diferentes aplicaciones se comuniquen entre sí. Es una interfaz que define cómo pueden interactuar los componentes de software. Las API se utilizan para solicitar y enviar datos entre diferentes sistemas, como aplicaciones web, servicios en la nube, bases de datos, etc. En este caso, estás utilizando una API para enviar los datos de un formulario desde tu página web al servidor.
